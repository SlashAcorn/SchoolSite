Why should I use Linux

Linux is great because it is free, open source, and secure. Linux being free means that you do not have to worry about licenses and deal with all the 'activate Windows' nonense. Linux is also open source which allows for immense customisability, security and freedom, when you use Linux you can be certain that the OS is not stealing your data because all of the source code contained in the OS is available for you to read. Linux is secure due to its open source nature, there are hundreds or even thousands of people actively helping patch security holes, this is much more than however many people are fixing Windows security holes (There is a reason the Windows server never took off).

Who uses Linux

Everyone uses Linux, including you! Linux runs on 92% of all the worlds servers meaning that if Linux stopped existing today, you would lose most of the internet. Linux also dominates the mobile market with 70% of phones running Android which is a derivative of Linux. The only market that Linux doesn't have the majority is the desktop market, and the Linux desktop is rapidly improving with 70% of games on Steam being playable through Proton, a compatibility layer that allows for Windows programs to run on Linux.
