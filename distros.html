What is a Linux Distrobution?

A Linux Distrobution or 'Distro' is a pre packaged system that uses Linux as its kernel. Most of these distros run other software developed by the GNU project alongside linux to allow for a functioning system, this is why some people refer to a Linux OS as GNU/Linux.

Summary of Popular Linux Distros

  • Arch Linux is a very popular choice because it allows for extreme customisability with its install proccess.
  • Linux Mint is popular among new users because it is easy to use and has a windows-like desktop environment.
  • Fedora Linux is a common Linux distro mostly used for its widespread support for applications.
  • Gentoo Linux is a hardcore Linux distro allowing for even more customisability than Arch because of the compiled install proccess.

What Distro is For Me?

There are many distrobutions of Linux suited for people who like to use their computer in different ways. One of the things that makes Linux so good is how adaptable it is for different needs. There are websites available that can help you decide on which distro to use like distrochooser.de

Hide DistroChooser

There is lots of misinformation and unhelpful information when it comes to chosing a Linux distro. Some blog posts will give contradictory information. The best thing to do is just to stay away from articles titled anything similar to "The Top 10 BEST Linux Distros in 2023".

One of the best ways to chose a distro is just to pick a common one for beginners like Mint, Fedora or Pop!_OS. Don't spend to much time on chosing the distro as you can always change later. Most aspects of different Linux distros are global accross all distros, for example Mint's Windows-like desktop en vironment can be installed on any other distro.
