Getting Started With Linux


What exactly is Linux?

Linux isn't technically an operating system as it is just the kernel. The kernel is the core part of software that makes up an operating system. The kernel allows for higher level software to interface with hardware like the CPU, memory and I/O.

Since Linux is just the kernel, you can not just install it and have a working system. To have a usable system you need the help of other tools that are curated for different needs, these are also known as distrobutions or distros.
