How to install Linux

Most Linux distrobutions have installer .iso files that you can flash to a USB stick using a program like Rufus or Etcher (I use Raspberry Pi Imager). Some distros have harder installations that require you to use the command line to manually install the system. There are distroubtions like Gentoo Linux that even require you to compile most software that you use manually which results in programs that run faster, but the hastle of compiling your programs manually is often too much for most people.

Arch Linux

Arch Linux is a distrobution well known because of its command line installation. It isn't as hard or time consuming as Gentoo and can be installed faster than Windows if you know what you are doing. The intimidating command line installation looks scary at first but isn't really that hard and mostly involves copying and pasting from the Arch Linux wiki. Even though the installation isn't that hard, it gets to the heads of some that use it which results in some Arch users being pretty elitist and egotistical, this is where the joke 'I use arch btw' comes from.
(I use arch btw)
